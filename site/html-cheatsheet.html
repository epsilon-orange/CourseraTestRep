Idiots Guide to HTML

Step One: Basic Setup

Step Two: Tags

  1. Basic Tags

    2. Tag Description
    <!DOCTYPE>Defines the doc type
    <html>Defines root of doc
    <head>Defines head of doc, contains non-visible meta data
    <body>Defines body of the doc, contains everything you see on webpage
    <h1>-<h6>Defines headings
    <p>Defines paragraphs
    <br>Defines line break
    <hr>Defines horizontal line
    <!--...-->Writing Comments

  2. Formatting Tags

    3. Tag Description
    <b>Bold
    <strong>Bold
    <i>Italic
    <em>Italic
    <u>Underline
    <sub>Subscript
    <sup>Superscript
    <sup>Superscript
    <pre>Pre formated text
    <small>Small text
    <abbr>Abbreviation
    <code>Used to display code
    <kbd>Displays keyboard input
    <del>Strikethrough
    <s>Strikethrough
    <ins>Underlines
    <mark>Highlight
    <blockquote>Use for quotes
    <address>Displays address or contact info
    <cite>Title of a cited work
    <q>Adds quotation marks

  3. Image Tags

    4. Tag Description
    <img>Defines image
    <figure>Defines figure, a container for image
    <figcaption>Define caption for figure
    <picture>Holds multiple sources for images

  4. Link or Nav Tags

    5. Tag Description
    <a>Hyperlink to webpage
    <link>Link to other documents
    <nav>Wraps nav links

  5. List Tags

    6. Tag Description
    <ol>Ordered list
    <ul>Unordered list
    <li>Item in list

  6. Form and Input Tags

    7. Tag Description
    <form>Defines form
    <input>Defines an input
    <label>Defines a label for input element
    <textarea>Defines an input section for text
    <select>contains <option> for the user to choose
    <optgroup>contains a group of options
    <option>Defines the users options to choose
    <fieldset>Defines a section of the form
    <legend>Defines the title of a section of a form
    <datalist>Defines a list of items users can chose from for an input
    <button>Defines a button

  7. Table Tags

    8. Tag Description
    <table>Defines table
    <thead>Groups the head content of the table
    <tbody>Groups the body content of the table
    <tfoot>Groups the footer content of the table
    <tr>Defines a row in the table
    <th>Defines a header cell in the table
    <td>Defines a cell in the table
    <caption>Defines the caption of the table
    <colgroup>Defines a column of the table
    <col>Give property to the column within the colgroup

  8. Media Tags

    9. Tag Description
    <audio>Audio content
    <video>Video content
    <track>Defines text track for the audio and video
    <source>Specifies resource from the media

  9. Meta Tags

    10. Tag Description
    <meta>Define metadata
    <link>Defines a base URL

  10. Style and Programming Tags

    11. Tag Description
    <style>Provides CSS for the HTML elements
    <script>Defines javascript program

  11. General Tags

    12. Tag Description
    <div>Defines a group of elements
    <span>Defines inline section
    <header>Defines header
    <footer>Defines footer
    <main>Defines main content
    <article>Defines article
    <details>Defines a section the user can view or hide
    <summary>Defines summary of details
